# Домашнее задание 2.

Установка и настройка React. JSX как основа ReactJS. Functional components

## Скрипты запуска

Доступные скрипты проекта

### `npm run start`

Запуск приложения
Open [http://localhost:8081](http://localhost:8081) в браузере.

Горячая перезагрузка страницы.\
Вывод ошибок в консоль.

### `npm run test`

Запуск тестов.

### `npm run lint`

Запуск линтера

### `npm run lint:fix`

Авто исправление линтером ошибок

### `npm run storybook`

Запуск storybook

### `npm run build-storybook`

Сборка storybook

### `npm run build`

Сборка приложения в каталог 'public'.

### `npm run chromatic`

Запуск chromatic.
